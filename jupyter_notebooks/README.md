# Jupyter Notebooks

These are the interactive notebooks used to take real measurements. Both walk through the same QICK-DAWG measurement workflow (PL intensity, ODMR, Rabi, Hahn echo, T1, Ramsey, etc.) and differ only in which detector they're set up for:

- **`APD_Notebook.ipynb`** — avalanche photodiode detector, read out as an analog signal.
- **`Photon_Counting_Notebook.ipynb`** — single photon detector, read out via edge counting.

Pick whichever matches your detector hardware; everything else in the notebook is the same.
